# Chess row color

Найдите цвет квадрата шахматной доски. Квадрат А1 черный.

```js
"black"  == solution("A", 1)
"black"  == solution("H", 8)
"black"  == solution("D", 4)
"black"  == solution("G", 7)
"white"  == solution("A", 8)
"white"  == solution("H", 1)
"white"  == solution("E", 8)
"white"  == solution("E", 4)
```
