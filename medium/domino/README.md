# Domino

x содержит максимальное количество точек на одном конце доминошки. Набор домино, это набор всех возможных комбинаций костяшек домино. Выведите количество точек в наборе домино. Например 2 -> 12, 1 -> 3 (0|1, 1|1)

Examples:
```js
12  == solution(2)
0  == solution(0)
3  == solution(1)
30  == solution(3)
```
