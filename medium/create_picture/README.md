# Create picture

Дан массив слов, напишите функцию, которая возвращает картинку (массив) с границей из звездочек *, внутри границы находятся самые длинные слова.

Examples

```js
["*****","*def*","*****"]  == solution(["a","bc","def"])
["******","*some*","*word*","******"]  == solution(["some","word","foo","bar"])
```
